生成静态页面

通过hexo g生成静态文件，会在根目录E:\Blog\下生成public文件夹，里面包含相应的html页面。

启动服务器

通过hexo s启动服务器，打开浏览器，输入网址http://localhost:4000/ 即可看到hexo生成的静态网页。

至此，hexo博客就搭建完毕。

应用Butterfly主题

在官网的主题页面选择自己心仪的主题，这里选择以Butterfly主题来应用

参考：Butterfly主题展示页面 官方的配置文档

主题安装

  • 下载主题

    hexo 4.2以上的系统需要安装cheerio

    npm install cheerio@0.22.0 –save

  • 安装pub和stylus渲染器

主题应用

  • 修改配置文件_config.yml，找到theme，把landscape修改成Butterfly，例如我的在E:\Blog\下，修改_config.yml即可

    推荐把主题默认的配置文件_config.yml复製到 Hexo 工作目录下的source/_data/butterfly.yml，如果source/_data的目录不存在那就创建一个。

    注意，如果你创建了butterfly.yml, 它将会替换主题默认配置文件_config.yml里的配置项 (不是合并而是替换), 之后你就只需要通过git pull的方式就可以平滑地升级 theme-butterfly了。

    刷新http://localhost:4000 ,可以看到更新后的主题。
